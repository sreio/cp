核心组件说明

前端 (Vue 3 + Vite)

  • 彩票信息展示（开奖号码、奖池、中奖详情）
  • 数据分析可视化（ECharts）
  • 推荐号码生成与分享
  • 响应式设计 + 暗黑模式

后端 (Hono on Worker)

  • RESTful API 服务
  • 定时任务（Cron Trigger）
  • 数据采集与存储
  • Webhook 通知服务

数据库 (D1 + SQLite)

  • 统一数据库适配器
  • 生产环境：Cloudflare D1
  • 开发环境：本地 SQLite
  • 自动迁移与版本管理

数据源

  • 官方 API：最新开奖数据
  • 第三方 API：历史数据补全
  • 智能去重与增量更新
  • 数据校验与异常处理
